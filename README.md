# Titanic Dataset - Data Cleaning & Preparation

## Project Overview

This project was completed as part of the DecodeLabs Data Analytics Internship.

The objective was to clean and prepare a raw dataset for analysis using Python and Pandas.

## Dataset

Titanic Dataset

## Tasks Performed

* Loaded the dataset using Pandas
* Explored the dataset structure
* Identified missing values
* Removed duplicate records
* Filled missing values
* Examined data types and column information
* Exported the cleaned dataset

## Tools Used

* Python
* Pandas
* Jupyter Notebook

## Files

* `cleaned_dataset.csv` - Cleaned dataset
* `Data_Cleaning_Titanic_Dataset.ipynb` - Jupyter Notebook containing the project code

## Learning Outcomes

Through this project, I learned:

* Data cleaning techniques
* Handling missing values
* Removing duplicate records
* Working with CSV files using Pandas
* Basic data preparation workflow
